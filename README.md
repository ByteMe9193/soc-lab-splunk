# SOC Lab — Splunk SIEM

This project documents the implementation of a small SOC laboratory using Splunk.

The lab simulates a monitoring pipeline where:

Kali Linux - generates attack activity  
Windows - records security events  
Splunk - collects and analyzes logs

## Architecture

- Kali Linux — attacker machine
- Windows — monitored endpoint
- Ubuntu Server — Splunk SIEM server

## Technologies Used (so far)

- Splunk Enterprise
- Splunk Universal Forwarder
- Windows Event Logs
- Ubuntu Server
- Kali Linux
